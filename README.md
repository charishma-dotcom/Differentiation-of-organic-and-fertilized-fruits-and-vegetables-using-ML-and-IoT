# 🍎 Differentiation of Organic and Fertilized Fruits and Vegetables Using ML and IoT

## 📌 Project Overview

The **Differentiation of Organic and Fertilized Fruits and Vegetables Using ML and IoT** project is an intelligent system that identifies whether fruits and vegetables are organic or chemically fertilized using Machine Learning, Deep Learning, Image Processing, and IoT technologies.

The system captures images of fruits and vegetables, processes them using trained machine learning models, and combines IoT sensor data to improve prediction accuracy. It provides users with reliable classification results through a simple and user-friendly interface.

---

## 🎯 Objectives

- Identify organic and fertilized fruits and vegetables.
- Improve food quality assessment using Artificial Intelligence.
- Integrate IoT sensor data with machine learning predictions.
- Provide accurate and real-time detection results.
- Maintain a history of previous predictions.

## 🤖 Machine Learning Models

This project uses:

- TensorFlow/Keras Deep Learning Model
- Scikit-learn Machine Learning Model
- OpenCV Image Processing
- Feature Extraction Techniques

The trained models analyze images and classify fruits and vegetables as either **Organic** or **Fertilized**.

---

## 🌐 IoT Integration

The IoT module collects sensor data that enhances prediction accuracy. The sensor values are processed along with image features to improve classification performance.

---

## 📊 Workflow

1. User logs into the system.
2. User uploads or captures a fruit/vegetable image.
3. IoT sensor data is collected.
4. Image preprocessing is performed.
5. Machine Learning model predicts the category.
6. Result is displayed.
7. Prediction history is stored in the database.

---

## 📈 Output

The application predicts whether the selected fruit or vegetable is:

- ✅ Organic
- ✅ Fertilized

The prediction result is displayed along with related information.
